## App Description

Expense Tracker is a simple single-resource CRUD web application that allows users to record, view, update, and delete expenses. Each expense includes a title, amount, description, and date. The application uses a React frontend, an Express/Node.js backend, and an in-memory array as the data storage. The main focus of this project is applying Test-Driven Development through unit, integration, and system tests.

## User Stories

1. As a user, I want to add an expense, so that I can record money I have spent.
2. As a user, I want to view my expenses, so that I can track my spending.
3. As a user, I want to edit or delete an expense, so that I can correct my records.

## Tech Stack

- Frontend: React JS
- Backend: Node.js and Express.js
- Data Storage: In-memory array
- Unit Testing: Jest
- Integration Testing: Jest and Supertest
- System/E2E Testing: Playwright
- CI/CD: GitHub Actions
